# Wordle Clone

A terminal-based Wordle game implementation in Python.

## Features

- 🎯 5-letter word guessing
- 🎨 Color-coded feedback (green, yellow, gray)
- 📝 Custom word list support
- 🔄 Play again option
- ⌨️ Keyboard interrupt handling
